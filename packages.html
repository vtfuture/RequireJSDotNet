RequireJS.NET packages hosted on NuGet.org

RequireJS.NET

Contains the ASP.NET MVC integration components, HTML helpers, filters and helper classes.

PM> Install-Package RequireJsNet

RequireJS.NET Compressor

MsBuild task that parses RequireJS.config files and builds JavaScript bundles using YUI compressor.

PM> Install-Package RequireJsNet.Compressor

RequireJS.NET ResxToJs

MSBuild task that outputs specified keys from a resx file in RequireJS i18n format.

PM> Install-Package RequireJsNet.ResxToJs

RequireJS.NET Scripts

Contains the minfy version of require.js the text and i18n plug-ins.

PM> Install-Package RequireJsNet.Scripts
